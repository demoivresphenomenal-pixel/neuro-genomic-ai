# Quick Start Guide - Neuro-Genomic AI Project

## 🚀 Get Started in 5 Minutes

### 1. Clone the Repository

```bash
# Using HTTPS
git clone https://github.com/demoivresphenomenal-pixel/neuro-genomic-ai.git
cd neuro-genomic-ai

# Or using SSH (if configured)
git clone git@github.com:demoivresphenomenal-pixel/neuro-genomic-ai.git
cd neuro-genomic-ai
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Open Jupyter Notebook

```bash
jupyter notebook notebooks/exploration.ipynb
```

### 4. Or Use Google Colab (No Installation Needed)

Open this URL:
```
https://colab.research.google.com/github/demoivresphenomenal-pixel/neuro-genomic-ai/blob/main/notebooks/exploration.ipynb
```
